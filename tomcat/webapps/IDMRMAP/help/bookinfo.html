Novell Identity Manager Role Mapping Administrator 4.0.2 User Guide

This guide explains how to use the Novell Identity Manager Role Mapping Administrator to map authorizations in managed systems to Identity Vault roles. It contains the following sections:

  • Getting Started

  • Mapping Roles

  • Managing Lists

  • Generating Reports

  • Troubleshooting

Audience

This guide is intended for any users responsible for establishing and maintaining cross-domain policy relationships between Novell Identity Manager and managed systems. To use the Role Mapping Administrator, you must be defined as a Roles Administrator or Roles Manager in the Identity Manager Roles Based Provisioning Module.
